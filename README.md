# Dataset Structure

Этот датасет содержит примеры 3D моделей и их ориентаций для обучения нейросети.

## Структура

- `models/` - Исходные STL файлы
- `orientations/` - Разные ориентации каждой модели
- `gcode_results/` - Результаты слайсинга

## Примечание

Полные STL и G-code файлы не загружаются на GitHub из-за размера.

## Формат данных

Каждая ориентация содержит:
- JSON файлы с метаданными
- Анализ геометрии
- Параметры печати

# G-code Results

Эта папка предназначена для результатов слайсинга (G-code файлов).

## Почему файлов нет на GitHub?

G-code файлы обычно очень большие (десятки мегабайт) и не должны храниться на GitHub.

## Структура (локально):
gcode_results/
├── default/
│ └── output.gcode # Пример: 15MB
├── flat/
│ └── output.gcode # Пример: 14MB
└── optimal/
└── output.gcode # Пример: 16MB
