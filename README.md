aa# Dataset Structure

Этот датасет содержит примеры 3D моделей и их ориентаций для обучения нейросети.

## Структура

- `models` - Исходные STL файлы
- `results` - Разные ориентации каждой модели
- `auto_analyze_full` - Скрипт для создания элементов выборки
- `unified_analyzer` - Скрипт для создания элементов выборки

## Формат данных

Каждая ориентация содержит:
- JSON файлы с метаданными
- Анализ геометрии
- Параметры печати
