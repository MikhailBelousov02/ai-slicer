# Dataset Structure

Этот датасет содержит примеры 3D моделей и их ориентаций для обучения нейросети.

## Структура

- `models/` - Исходные STL файлы
- `orientations/` - Разные ориентации каждой модели
- `gcode_results/` - Результаты слайсинга

## Примечание

Полные STL и G-code файлы не загружаются на GitHub из-за размера.

## Формат данных

Каждая ориентация содержит:
- JSON файлы с метаданными
- Анализ геометрии
- Параметры печати
